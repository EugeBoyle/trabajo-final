Este sistema tendra como objetivo generar reportes estadisticos sobre los examenes redidos de diferentes cursos en linea,
 dandole informacion valiosa al profesor para tomar decisiones, o saber cuanto exito tuvo el examen que elaboro.


 el sistema, permite a un profesor:
 1. cargar, modificar y eliminar datos de sus alumnos. 
 2. cargar, modificar y eliminar cursos.
 3. cargar examenes con nota, curso y alumno.
 4. generar reporte estadistico para los distintos cursos
